# フリーランス転向の実践ステップ

## 転向前チェックリスト

フリーランスに転向する前に確認すべき項目：

- [ ] 経験年数3年以上（実務経験）
- [ ] 貯金6ヶ月分以上
- [ ] 確定申告の基礎知識
- [ ] 開業届の準備
- [ ] クレジットカード・ローンの整理（独立前に審査を通す）
- [ ] 健康保険・年金の切り替え計画
- [ ] 最低1つのエージェントに登録

## Step 1: エージェント選び

フリーランスエージェントの選び方：

| 評価軸 | 確認ポイント |
|--------|------------|
| マージン率 | 公開しているか、何%か |
| 案件数 | 自分のスキルセットに合う案件があるか |
| 支払いサイト | 月末締め翌月払いか、翌々月か |
| サポート | 確定申告支援、福利厚生 |
| 契約形態 | 準委任か請負か |

## Step 2: 案件獲得

1. エージェント経由の案件応募
2. 知人・元同僚からのリファラル
3. 技術コミュニティでの人脈形成
4. SNS・ブログでの情報発信

## Step 3: 初月の立ち上げ

- 開業届を税務署に提出
- 青色申告承認申請書を提出
- 事業用銀行口座の開設
- 会計ソフトの導入（freee、マネーフォワード等）

## この章のまとめ

フリーランス転向は準備が全てです。次章では、契約時に知っておくべき法律知識を解説します。
