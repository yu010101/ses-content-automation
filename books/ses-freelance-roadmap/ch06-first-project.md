# 最初のフリーランス案件で成功する方法

## 初案件で意識すべき3つのこと

### 1. 信頼構築を最優先
最初の案件で最も重要なのは、技術力の証明ではなく信頼の構築です。

- 報連相を徹底する
- 納期を必ず守る（バッファを持つ）
- 分からないことは早めに確認する

### 2. ドキュメントを残す
フリーランスとしての評価は、次の案件獲得に直結します。

- 作業ログを毎日記録
- 技術的な判断の理由を文書化
- 引き継ぎ資料を丁寧に作成

### 3. 次の案件を見据える
フリーランスは常に次の案件を考える必要があります。

- 契約終了の2ヶ月前から次の案件を探す
- 現場での人脈を大切にする（リファラルの源泉）
- 成果をポートフォリオに追記する

## フリーランス初年度の確定申告

| 項目 | 内容 |
|------|------|
| 申告方式 | 青色申告（65万円控除） |
| 経費例 | PC、書籍、交通費、家賃按分、通信費 |
| 会計ソフト | freee、マネーフォワード、弥生 |
| 申告時期 | 2月16日〜3月15日 |
| 必要書類 | 収支内訳書、確定申告書B |

## 継続的な単価アップ戦略

1. **半年ごとに市場価値を再評価** - エージェントに相談、他案件の単価を調査
2. **スキルの掛け算** - バックエンド × インフラ、開発 × PM など
3. **直請け案件の割合を増やす** - マージンを減らし手取りを最大化
4. **発信を続ける** - ブログ、SNS、登壇で認知度を上げる
